# Bike Demand Prediction using Machine Learning

This project solves a real-world problem in bike-sharing systems by predicting hourly bike demand using machine learning techniques.

## Problem

Bike-sharing systems often face imbalance issues where some stations run out of bikes while others remain underutilized.

## Solution

A machine learning model is built to predict the number of rented bikes at any given hour.

## Dataset

The dataset used in this project is the Seoul Bike Sharing Demand dataset from Kaggle:
https://www.kaggle.com/datasets/saurabhshahane/seoul-bike-sharing-demand-prediction

## Features

* Temperature, Humidity, Rainfall
* Hour of the day
* Season
* Holiday vs Working Day
